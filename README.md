# Weather Data API Application

## Overview

The Weather Data API Application is a Python project that retrieves historical weather data using a weather API, stores the results in a SQLite database, performs basic data analysis, and includes automated unit tests. This project demonstrates API integration, database management, data analysis, and software testing using Python.

---

## Features

- Retrieve historical weather data from a weather API
- Store weather data in a SQLite database
- Query and analyze stored weather information
- Calculate weather statistics
- Automated unit testing with Python's unittest framework

---

## Technologies Used

- Python 3
- SQLite3
- Open-Meteo API
- Requests
- unittest

## Installation

Clone the repository:

```bash
git clone https://github.com/joshreisinger17-Data/weather-data-api-application.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

Run the unit tests:

```bash
python test_weather.py
```
